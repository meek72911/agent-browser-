Privacy Policy

Last updated: April 2026

Overview

VibeStudio is designed with privacy as a core principle. We collect minimal data and all browsing activity stays local on your machine.

Data We Do NOT Collect

  • We do not collect browsing history
  • We do not collect page content or extracted markdown
  • We do not track which websites you visit
  • We do not collect personal information without consent
  • We do not send any telemetry data by default

License Key Validation

When you enter a license key, we send only a SHA-256 hash of the key to our validation server. The plaintext key is never transmitted or stored on our servers. Validation requests include only the key hash and your plan type.

Local Data Storage

All browsing data, settings, download history, and recent sites are stored locally on your device using the browser's localStorage. This data never leaves your machine. You can clear all local data from Settings > Privacy > Clear Browsing Data.

Research Extractions

When VibeStudio extracts page content for MCP (Model Context Protocol) operations, all processing happens locally. Extracted markdown is passed directly to the requesting IDE via local WebSocket connection. We never store or transmit extracted content to our servers.

Network Requests

VibeStudio makes the following external requests:

  • Website navigation (user-initiated)
  • Favicon loading via Google's favicon service
  • License key validation (hash only, on activation)
  • Update checks (version number comparison only)

Ad and Tracker Blocking

VibeStudio blocks ads and trackers by default using locally-stored blocklists. No data about blocked requests is sent externally.
